# Check if API access is available for a GWAS catalog study

Check if API access is available for a GWAS catalog study

## Usage

``` r
check_rest_avail(study_id)
```

## Arguments

- study_id:

  a study accession ID, e.g. "GCST000001"

## Value

TRUE if the REST API is available for the study, FALSE otherwise
