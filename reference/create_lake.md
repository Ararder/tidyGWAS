# Create a data lake in hivestyle format

Create a data lake in hivestyle format

## Usage

``` r
create_lake(dir, lake)
```

## Arguments

- dir:

  a directory containing tidyGWAS cleaned summary statistics. Each
  directory should contain a tidyGWAS_hivestyle directory.

- lake:

  a directory to create the data lake in.
